About

Welcome to 19th Century Map Maker! This website provides you with a unique way to interact with historical maps from the 1800s. Whether you're a history enthusiast, a researcher, or simply curious about the past, this platform allows you to explore this map in detail.

Our goal is to make historical maps more accessible and engaging.

Features:

  • Add pins to mark locations of interest
  • Customize pins with titles, descriptions, and colors
  • Export and embed created maps on other websites

If you have any suggestions, improvements, or feature requests, feel free to open an issue or create a pull request on GitHub. Happy exploring!

Help

Adding pins:

Required fields: Title
Optional fields: Description and Color (Color defaults to black if not specified)

Enhanced text:

To include enhanced text in your titles and descriptions, use HTML formatting elements. Here are helpful articles on text formatting and hyperlinks.

Removing pins:

Right click on a pin to remove it.

Exporting and Embedding

Copy the code under "Export" and paste it as an embed in another site. If needed, edit the embed code (i.e. change the center point, zoom level, or height of map)
