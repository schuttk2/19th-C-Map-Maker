About

This is an interactive map web application created by Kiley Schutte. The site uses the Leaflet JavaScript library. The purpose of this site is to provide an option for people wanting to create custom historical maps. The background map tile is Le Monde, principales decouvertes (1862), by F. A. Garnier (1803-1863). The map is available in the David Rumsey Map Collection.

Adding pins:

Optional fields: Description and Color (Color defaults to black if not specified)
